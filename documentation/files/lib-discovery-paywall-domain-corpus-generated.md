# File: `lib/discovery/paywall-domain-corpus.generated.json`

## Purpose
Stores the vendored domain corpus extracted from a local `bypass-paywalls-chrome-clean` checkout for reproducible discovery filtering.

## Contents
- `defaultDomains`: normalized domains from `sites.js`
- `updatedDomains`: normalized domains from `sites_updated.json`
- `customDomains`: normalized domains from `custom/sites_custom.json`
- `combinedDomains`: deduped union across all three sources
- `counts`: summary counts for each bucket

## Generation
- Generated by `scripts/extract-bpc-paywall-domains.mjs`.
- The source extension checkout is not required at runtime; only regeneration needs it.
