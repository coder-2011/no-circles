# TODO.md Review Prompt

Before merge or release, review root `TODO.md` as the active backlog.

Checklist:
1. Confirm all in-scope items are checked off or explicitly left open.
2. Separate true blockers from safe follow-up work.
3. Add immediate follow-up tasks for any residual risks discovered in review.
4. Keep completed items checked rather than deleting them during the active branch lifecycle.

Output format:
- completed items
- remaining items
- blockers, if any
- next 1-3 tasks
